# Stream Connect

This is a **mini full-stack project** created and completed by **Pavan R**.  

## Project Description
Stream Connect is a web application that allows users to join and manage video meetings. Users can create meetings, share meeting codes, and allow others to join easily.  

## Technologies Used
- **Frontend:** HTML, CSS, JavaScript, React  
- **Backend:** Node.js, Express.js, MongoDB (or your backend stack)  
- **Other:** JWT for authentication, Nodemailer for email notifications, CORS, dotenv
